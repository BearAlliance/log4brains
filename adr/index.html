    Architecture knowledge base

    Welcome to the project architecture knowledge base. This website tracks architecture decisions (also known as ADR - Architecture Decision Record), in a chronological order.

    Every ADR has one of these statuses:

    • Draft
    • Proposed
    • Accepted
    • Rejected
    • Superseded
    • Deprecated
